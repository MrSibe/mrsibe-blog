## 开发工具

- **VSCode**：开发必备代码编辑器，不用多说
- **Google Chrome**：不用多说，装机必备
- **ITerm 2**：MacOS 最好的终端
- **OrbStack**：轻量化 Docker 管理，体验比官方的 Docker Desktop 好不少。还支持创建 Linux 虚拟机和 k8s 管理
- **CC Switch**：灵活切换 Claude/Codex 等编程大模型 API

## AI 助手与知识管理

- **Alma**：强大的 Agent 客户端。我很喜欢把它当作助理
- **Cherry Studio**：功能非常多的大模型客户端，我之前特别喜欢用
- **Obsidian**：笔记软件，插件非常强大。折腾过笔记软件的肯定都用过
- **极速说**：语音编程神器，解放双手

## 系统增强与效率工具

- **Raycast**：万能启动器，最好的 Spotlight 替代品
- **鼠须管 (Rime)**：极致自定义的输入法（通过 iCloud 解决跨设备同步）
- **Ice**：极简的顶栏图标管理
- **BetterDisplay**：显示器管理，外接显示器开启 HiDPI 的刚需
- **Stats**：实时监控 CPU/GPU/网络状态，开源的很好用
- **AlDente**：通过限制充电阈值来延长电池寿命
- **PearCleaner**：轻量、开源的软件卸载残余清理工具。卸载的时候一定要再三确认！

## 办公与影音娱乐

- **OnlyOffice**：兼顾美观与兼容性的开源 Office 方案
- **IINA**：非常强大的播放器
